📖 Installer sur Android — Guide Complet
🎯 En suivant ce guide, l'application s'installe sur votre téléphone comme une vraie app — gratuitement, sans Play Store !
1

Créer un compte GitHub gratuit

Allez sur github.com → Cliquez "Sign up" → Entrez votre email, un mot de passe, un nom d'utilisateur → Vérifiez votre email.

🌐 github.com → bouton vert "Sign up" en haut à droite
2

Créer un nouveau dépôt (repository)

Une fois connecté → Cliquez le "+" en haut à droite → "New repository" → Nom : rapprochement → Cochez "Public" → Cliquez "Create repository".

➕ New repository → Name: rapprochement → ⚪ Public → Create repository
3

Télécharger et uploader le fichier

1. Téléchargez d'abord le fichier rapprochement.html depuis ce chat.
2. Dans votre dépôt GitHub, cliquez "Add file" → "Upload files".
3. Glissez le fichier rapprochement.html.
4. IMPORTANT : Renommez-le en index.html (cliquez sur le nom du fichier pour renommer).
5. Cliquez "Commit changes".

Add file → Upload files → Glisser rapprochement.html → Renommer en index.html → Commit changes
4

Activer GitHub Pages (mise en ligne gratuite)

Dans votre dépôt → Cliquez "Settings" (⚙️ en haut) → Faites défiler jusqu'à "Pages" dans le menu gauche → Sous "Branch", sélectionnez main → Cliquez "Save".

Attendez 2 minutes → Votre app sera disponible à :
https://VOTRE_NOM.github.io/rapprochement

Settings → Pages → Branch: main → Save → Attendre 2 min
5

Installer sur Android (comme une vraie app !)

Sur votre téléphone Android :
1. Ouvrez Chrome
2. Allez sur votre URL : https://VOTRE_NOM.github.io/rapprochement
3. Appuyez sur les 3 points (⋮) en haut à droite
4. Appuyez "Ajouter à l'écran d'accueil"
5. Confirmez → L'icône apparaît sur votre écran d'accueil ! 🎉

Chrome → ⋮ → "Ajouter à l'écran d'accueil" → Ajouter
✅ L'application fonctionne même sans connexion internet une fois installée !
Tout fonctionne directement sur votre téléphone : import, calcul, export Excel.
